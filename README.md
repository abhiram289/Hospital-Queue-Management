#  Hospital Queue Management System

A simple console-based C project that simulates a hospital queue management system.  
It allows users to register patients, manage their priority, and display the current queue in real time.

##  Features
- Add new patients with name, age, and ailment.
- Assign and manage queue priority.
- Display the current queue in order of priority.
- Simple and fast command-line interface.

##  Languages Used
- C 

##  How to Build and Run
1. Clone the repository:
   ```bash
   git clone https://github.com/abhiram289/Hospital-Queue-Management.git
   ```
2. Navigate into the folder:
   ```bash
   cd Hospital-Queue-Management
   ```
3. Compile:
   ```bash
   gcc main.c patient.c -o hospital.exe
   ```
   or 
   ```bash
   mingw32-make
   ```

4. Run:
   ```bash
   hospital.exe
   ```
